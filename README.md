# Gym Website

A modern gym/fitness website built with Next.js, TypeScript, and Tailwind CSS.

## Features

- Responsive hero section with animations (Framer Motion)
- BMI Calculator
- AI Workout recommendations
- Class booking modal
- Membership plans
- Trainer profiles
- Testimonials & gallery

## Tech Stack

- **Framework:** Next.js
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animation:** Framer Motion
- **Icons:** Lucide React

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it.
